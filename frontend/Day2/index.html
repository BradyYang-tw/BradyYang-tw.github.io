成為前端工程師-Day2之建立開發環境

前端工程師養成

Posted by     "Brady Yang" on Thursday, June 30, 2022

工欲善其事，必先利其器。要開始學習前端工程師的技能，首先就是建立開發環境，以提升自己的工作效率。

決定要當個前端工程師，但是要怎麼開始?

首先，很開心大家加入前端工程師的行列，不過各位是否發現到一個問題，我要在哪邊開始寫程式呢? 總不可能拿著紙筆寫吧。 在這邊我就向大家介紹一下，要在哪裡開始開發程式。

筆記本

若要進行前端開發的話，Javascript不需要像Python、Java去安裝軟體。我們可以於筆記本直接寫程式，並透過瀏覽器去檢查是否正確。 通常會搭配HTML呈現其效果。簡單來說，大家在網頁上看到的漂亮的動畫效果，都是透過Javascript呈現。 這部份我們之後開始練習Javascript時，都會在講解一遍，大家可以先有個印象就好。

線上編輯器

現在也有線上編輯工具，可以直接在線上進行開發。如

  • Codepen
  • JSFiddle

這樣的好處就是今天無論你在哪裡，你只要能夠連網就可以開啟你的專案在線上進行開發。

IDE

IDE又稱為整合開發環境(Integrated Development Environment)，其提供了許多功能可以讓開發者更加方便。 比如，今天你要看一個語法正不正確，IDE會提供註解。當你要設定顏色，IDE也可以直接顯示顏色讓你方便挑選。

常用的IDE為Visual Studio Code。下載完後，可以透過左側的Extension安裝一些PlugIn來加速開發的效率。

比如，你可以搜尋Prettier - Code formatter並安裝，這樣在開發時他就會幫你調整程式的格式排版。

總結

雖然上述方式都可以建立前端開發環境，但我個人還是建議透過IDE搭配瀏覽器做開發。 畢竟有需多方便的PlugIn可以使用。藉此提升開發效率。

總之，開發環境已經建立完成，下一章我們將開始學習HTML。
